# Tu Marca Eres Tú · I Congreso La Salle Business School Alliance

Diagnóstico interactivo de fortalezas y estilo de comunicación para el taller del **7 de abril de 2025**, 8:00–9:00 pm.

## URL pública
👉 **https://jfrubiano72.github.io/congreso-lasalle/**

## Qué hace la app
- 20 preguntas en 5 bloques: Fortalezas, Comunicación, Marca Personal, Neuromarketing, Cierre
- Resultados personalizados: 3 fortalezas dominantes + estilo de comunicación + tips accionables + frase de marca borrador
- Panel del facilitador: resultados grupales en tiempo real (localStorage), con 3 pestañas: Estilos · Fortalezas · Ventajas de cada estilo
- Funciona 100% en el navegador del celular — sin servidor, sin instalación

## Cómo usar en el taller
1. Proyectar el QR code en pantalla → participantes abren la URL en su celular
2. Mientras responden, abrir el **Panel del facilitador** en el computador conectado al proyector
3. El panel muestra en tiempo real la distribución de estilos e insights para el facilitador

## Stack
HTML + CSS + JavaScript vanilla · localStorage para datos grupales · GitHub Pages para hosting
